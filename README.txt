ความสามารถของระบบจำแนกตามกลุ่มที่ใช้งานออกเป็น 2 ส่วนดังนี้ 
1. ผู้ดูแลระบบ คือ เจ้าหน้าที่ดูแลระบบ 
	- สามารถเข้าสู่ระบบได้
	- สามารถเพิ่ม ลบ แก้ไข ข้อมูลสินค้า
	- สามารถปรับเปลี่ยนสถานะการสั่งซื้อสินค้าได้
	- สามารถสรุปรายงานการสั่งซื้อสินค้าได้
	- สามารถตรวจสอบสินค้าคงเหลือได้
2. ลูกค้า
	- สามารถสมัครสมาชิกได้
	- สามารถเข้าสู่ระบบได้
	- สามารถค้นหาข้อมูลสินค้าได้
	- สามารถเพิ่ม ลบ แก้ไข และยกเลิกรายการสั่งซื้อได้
	- สามารถสั่งซื้อสินค้าได้
	- สามารถแจ้งการชำระเงินได้
	- สามารถตรวจสอบสถานะการสั่งซื้อได้
